# 🎨 Emora

A fun and interactive web application where you can draw emojis based on your current feelings and emotions!

## Features

✨ **Express Yourself**: Select from 8 different emotions and draw your interpretation
🎨 **Customizable Drawing**: Adjust brush size and choose any color you want
📥 **Download**: Save your creations as PNG images with timestamps
📱 **Mobile Friendly**: Works on desktop, tablet, and mobile devices
⚡ **No Dependencies**: Pure HTML, CSS, and JavaScript

## Emotions Available

- 😊 Happy
- 😢 Sad
- 😠 Angry
- 😲 Surprised
- 😎 Cool
- 😍 In Love
- 😕 Confused
- 😴 Tired

## How to Use

1. Open `index.html` in your web browser
2. Select your current emotion from the dropdown menu
3. Use the brush size slider to adjust how thick your strokes are
4. Pick a color using the color picker
5. Draw your emoji on the canvas
6. Click "Download Emoji" to save your artwork
7. Click "Clear Canvas" to start over

## Installation

Simply clone this repository and open `index.html` in your favorite browser:

```bash
git clone https://github.com/587Algo/emotion-emoji-drawer.git
cd emotion-emoji-drawer
# Open index.html in your browser
```

Or deploy it to GitHub Pages or any static hosting service!

## Files

- `index.html` - Main HTML structure
- `styles.css` - Beautiful styling with gradient backgrounds
- `script.js` - Canvas drawing logic and event handlers
- `README.md` - This file

## Customization

You can easily customize:
- Add more emotions by editing the `emotionEmojis` object in `script.js`
- Change the color scheme in `styles.css`
- Modify canvas size by changing `canvas.width` and `canvas.height` in `script.js`

## Browser Support

Works on all modern browsers that support:
- HTML5 Canvas
- ES6 JavaScript
- CSS Grid
